# Den's Portfolio

A personal developer portfolio built with React and Vite to showcase projects, skills, and contact information.

## Features

- Simple, responsive single-page layout
- Sections for Hero, About, Projects, and Contact
- Reusable UI components for projects and skill bars
- Intersection observer hook for subtle animations

## Tech Stack

- React (JSX)
- Vite
- Plain CSS (modular files in `src`)

## Getting Started

1. Install dependencies:

```bash
npm install
```

2. Run development server:

```bash
npm run dev
```

3. Build for production:

```bash
npm run build
```

4. Preview production build locally:

```bash
npm run preview
```

## Project Structure (high level)

- `src/` — app source code (components, sections, UI, hooks, data)
- `public/` — static assets
- `index.html` — app entry
- `vite.config.js` — Vite configuration
